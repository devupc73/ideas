# Roles del sistema (MVP)

## Capitán
Responsable de organizar el partido semanal.

Permisos:
- crear partido
- ver jugadores registrados
- confirmar pagos
- ejecutar sorteo de equipos
- administrar lista de espera

## Jugador
Participante del grupo.

Permisos:
- registrarse al partido
- pagar su cupo
- ver equipos asignados
