# Piloto actual

## Configuración operativa
- Día fijo: miércoles
- Hora fija: 22:00
- Modalidad: 9v9
- Capacidad: 18 jugadores
- Tipo: partido recurrente semanal

## Reglas
1. El sistema genera el partido semanal automáticamente.
2. Solo cuentan para cupos los jugadores con pago confirmado.
3. Del jugador 19 en adelante, se crea lista de espera.
4. El sorteo automático arma dos equipos de 9 jugadores.
5. Si alguien se retira, sube el primero de la lista de espera.
6. El excedente de recaudación se registra como saldo del grupo.
